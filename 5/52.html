OWASP ZAP

OWASP ZAP is a web application scanner based on the DAST (Dynamic Application Security Testing) methodology. The methodology allows you to detect security issues in a running application or website by scanning them for known vulnerabilities. Such vulnerabilities include SQL injections, cross-site scripting (XSS), Clickjacking, etc.


OWASP ZAP is developed and supported by the project of the same name called OWASP (Open Web Application Security Project) - a non-profit organization that specializes in creating articles, materials, documentation, tools and technologies that allow you to develop applications more securely, as well as ensure the proper level of information security for existing applications and websites.


The advantages of OWASP ZAP include:

Cross-platform - support for all major operating systems (Windows, Linux, MacOS);

Free open source project;

Support for plugins to expand functionality;

Ability to work both through a graphical interface (GUI) and through a command line interface;

A wide range of functions — from active/passive scanning to API and AJAX scanning;


Ease of use. Ideal for both beginners in information security and professionals.


Installation

OWASP ZAP is cross-platform software and can be installed on Windows, Linux and MacOS. It can also be launched in a Docker container. To install and run the program on Windows and Linux, you need pre-installed Java version 11 or higher. The installer for MacOS already includes Java. To run in a Docker container, you also do not need pre-installed Java. The official website contains installers and instructions for each OS.

For Docker, images with the following tags are presented separately: Stable — standard (regular) version; Bare — minimal version. Contains only the most necessary functions. Ideal for running in the command line; Weekly — version, updates for which are released every week; Live — beta version. Contains the latest release.


Operating principle

When scanning, ZAP creates its own proxy server through which all scanning requests are processed. ZAP includes special search robots (crawlers) that perform vulnerability identification. The proxy server is located between the user's browser and the end web application.

Burp Suite

Burp Suite is a tool for web application security audit. It contains tools for web application mapping, file and folder search, query modification, fuzzing, password guessing, and much more. There is also a BApp store, which contains additional extensions that increase the functionality of the application. Designed to conduct an audit of a web application, both in manual and automatic modes.


The tool itself is a proxy mechanism that intercepts and processes all requests coming from the browser. It is possible to install a burp certificate to analyze https connections.


There are two versions of Burp Suite: Professional and Free.


The free version is a full-fledged testing tool. One of the main differences is the lack of a scanner in the free version and a limit on the number of requests per unit of time. Also, the free version does not have built-in payloads for the intruder, but you can use external ones, for example: github.com/1N3/IntruderPayloads. There are also restrictions on the use of add-ons from the BApp store. A significant disadvantage is the lack of Burp Collaborator in the free version (using an external service to identify vulnerabilities).


The main functionality is based on the following modules:

Proxy — an intercepting proxy server operating over the HTTP(S) protocol in man-in-the-middle mode. Situated between the browser and the web application, it will allow you to intercept, study and modify traffic going in both directions.

Spider — a spider or crawler that allows you to automatically collect information about the architecture of a web application.

Scanner — an automatic vulnerability scanner (OWASP TOP 10, etc.) Available in the Professional version, the free version only has a description of the capabilities. Intruder — a utility that allows you to automatically perform various types of attacks, such as password guessing, identifier enumeration, fuzzing, etc.

Repeater — a utility for modifying and re-sending individual HTTP requests and analyzing application responses.

Sequencer — a utility for analyzing the generation of random application data, identifying the generation algorithm, and data predictability.

Decoder is a utility for manual or automatic conversion of web application data.

Comparer is a utility for identifying differences in data.

Extender is an extension in BurpSuite. You can add both ready-made ones from the BApp store and your own development.
